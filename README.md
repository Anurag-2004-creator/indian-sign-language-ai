# Indian Sign Language AI

## Overview
This project is an AI-based system that recognizes Indian Sign Language (ISL) gestures using computer vision and machine learning. It aims to help bridge the communication gap for the hearing and speech-impaired community.

## Features
- Real-time hand gesture capture using webcam
- Dataset creation for ISL gestures
- Machine learning model training
- Gesture prediction from live video feed

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
- scikit-learn
